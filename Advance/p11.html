css pseudo-elements...!

the css pseudo-element allow you to style the elements or parts of an elements without adding any ids or classes to them.

css3 introduced a new double-colon(::) syntax for pseudo-elements to distinguish between them and pseudo classes.

selector::pseudo-element{property: value:}
