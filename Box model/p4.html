CSS BOX MODELL..!

Every element that can be displayed on a web page is compired of one or more rectangulars boxes.CSS box model typically describes how these rectangular boxes are laid out on a web page.These boxes can have different properties and can interest with each others in different ways.but every box has a content area and optional surrounding padding,border,and margin areas.
